# Ground Station Hardware

This is a Pico 2W with the following pinout:

| Pin |      Function         |
| --- | --------------------- |
| 1   | Zenith Servo          |
| 2   | Azimuth Servo         |
| 11  | GPS RX (UART 1 TX)    |
| 12  | GPS TX (UART 1 RX)    |
| 14  | LoRa RST              |
| 15  | LoRa G0               |
| 24  | SPI0 SCK (LoRa SCK)   |
| 25  | SPI0 RX (LoRa MISO)   |
| 26  | SPI0 TX (LoRa MOSI)   |
| 27  | SPI0 CS (LoRa CS)     |
